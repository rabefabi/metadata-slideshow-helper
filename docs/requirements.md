# User Requirements for this Integration

- Publishable via HACS as a Home Assistant custom integration.
- Provide a slideshow usable in dashboards, cycling images automatically.
- Allow selecting a media directory inside Home Assistant.
- Support metadata filtering: include/exclude tags, minimum star rating.
- Expose image(s) to Lovelace picture card; aim to avoid custom frontend.
- Keep backend Python-only approach, don't add any extra services
